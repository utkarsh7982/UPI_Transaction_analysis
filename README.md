This project provides a complete analysis of UPI (Unified Payments Interface) transactions using Excel, SQL, Python, and Power BI.
The goal is to understand transaction trends, app-wise performance, success & failure rates, and overall digital payment patterns.

The analysis provides:
-Monthly transaction trends
-App-wise transaction comparison (PhonePe, Google Pay, Paytm, etc.)
-Total transactions, successful transactions, and failed transactions
-Success rate & failure rate KPIs
-Insights using an interactive Power BI dashboard
-Data cleaning and preprocessing steps

Technologies Used:
-Excel – Initial data cleaning
-SQL – Data transformation and queries
-Python – EDA + Data preprocessing
-Power BI – Final dashboard & insights

Key Dashboard Features:
-Trend line for monthly UPI transactions
-App-wise bar chart
-Success vs Failure rate card visuals
-Transaction distribution
-Interactive slicers for better filtering
